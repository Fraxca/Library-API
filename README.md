-🚀 Features
- 🔍 Retrieve all books
- 📖 Get book details by ID
- ✏️ Update book information
- ➕ Add new books
- 🗑️ Delete existing books
- 💻 RESTful interface with Flask
- 📂 Persistent storage with SQLite

- 💻 Usage
- Run main.py  
- Access endpoints (/livros, /livros/<id>)
- Send requests via GET, POST, PUT, DELETE
- Receive results in JSON format for easy integration
